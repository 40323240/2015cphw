上課筆記

1.至 Leo-editor GitHub
2.點選 releases 下載最新的zip檔 (broke-abbrev)
3.直接將載下來的檔案拉到 Ana3_2015 的 data
4.至上課業面
5.點選左側欄位的 2015fall課程
6.下載 fix資料夾拉到Ana3_2015
7.開啟SciTE
8.將fix資料夾內的fix.bat及start.bat拉入SciTE
9.fix.bat需新增 copy leo-editor-broke-abbrev.zip E:\Ana3_2015\data\ (老師的影片是放在C槽，我的是放在E槽的隨身碟)
10.start.bat把第25行改為 pip install leo-editor-broke-abbrev.zip 點選View → Line Numbers 可以方便找到25行
11.儲存完之後點選fix.bat
12.再啟動 start 就安裝了！
